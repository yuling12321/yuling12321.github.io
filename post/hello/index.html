    HTML学习笔记

    1. HTML基础

    1.1 什么是HTML?

    HTML 是用来描述网页的一种语言。

    • HTML 指的是超文本标记语言: HyperText Markup Language
    • HTML 不是一种编程语言，而是一种标记语言 (markup language)
    • 标记语言是一套标记标签 (markup tag)
    • HTML 使用标记标签来描述网页
    • HTML 文档包含了HTML 标签及文本内容
    • HTML文档也叫做 web 页面
    • “超文本”就是指页面内可以包含图片、链接，甚至音乐、程序等非文字元素
    • HTML文件后缀为.html或者.htm（推荐使用.html）
